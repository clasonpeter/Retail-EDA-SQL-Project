# Retail-EDA-SQL-Project

Online Retail Sales Exploratory Data Analysis

Overview:

This project performs comprehensive exploratory data analysis (EDA) on online retail transaction data using MySQL. The analysis examines customer behavior, product performance, sales trends, return patterns, and geographic distribution to support data-driven business decisions.

Analysis Sections

0. Data Quality Check

    - Validates data completeness
    - Counts total rows, customers, and products
    - Samples data for verification

1. Core Sales Metrics

    - Total Revenue: Overall business performance
    - Total Quantity Sold: Volume metrics

2. Product Performance

    - Top 10 Products by Revenue: Highest-earning items
    - Top 10 Products by Quantity: Best-selling items by volume

3. Customer Analysis

    - Top 10 Most Valuable Customers: Identifies high-value customers (LTV metric)
    - Orders per Customer: Purchase frequency analysis

4. Time-Based Analysis

    - Monthly Revenue Trend: Tracks sales performance over time
    - Day of Week Patterns: Identifies peak shopping days

5. Returns & Refunds Analysis

    - Transaction Type Summary: Separates purchases from returns
    - Top Return Products: Items with highest return values

6. Geographic Analysis

    - Revenue by Country: Market performance by region

7. Final Quality Check

    - Missing Customer IDs: Data completeness validation
    - Return Transaction Count: Scale of returns
  
# Key Insights

This analysis helps answer critical business questions:

- Which products drive the most revenue?
- Who are our most valuable customers?
- What are our peak sales periods?
- Which products have high return rates?
- Which geographic markets are strongest?

Usage:

Execute each section sequentially or run specific queries as needed. The script is organized into logical sections for easy navigation and modification.
